======================================================
README - Village Numérique Résistant (Projet NIRD)
======================================================

Ce projet est la ressource éducative et de sensibilisation pour le **Numérique Inclusif, Responsable et Durable (NIRD)**. 
Il vise à promouvoir l'utilisation de solutions Open Source et l'éco-conception logicielle.

------------------------------------------------------
1. STRUCTURE DU PROJET
------------------------------------------------------

Le projet est une application web statique (HTML, CSS, JavaScript) simple et performante.

/
├── accueil.html        (Page d'accueil et présentation des principes NIRD)
├── contenu.html        (Détails sur les outils libres et les licences)
├── jeux.html           (Page du Quiz interactif)
├── recherche.html      (Moteur de recherche filtrable des ressources)
├── style.css           (Feuille de style CSS complète)
├── script.js           (Logique JavaScript pour le Quiz et le Moteur de Recherche)
└── logo/
    └── votre-logo.webm (Logo animé de l'en-tête - format WebM)


------------------------------------------------------
2. MISE EN PLACE (POUR DÉMARRER LE SITE)
------------------------------------------------------

1.  **Arborescence :** Assurez-vous que tous les fichiers HTML, CSS et JS sont à la racine (dossier principal), et que le fichier vidéo `votre-logo.webm` est bien dans le sous-dossier `logo`.
2.  **Lancement :** Ouvrez simplement le fichier `accueil.html` avec n'importe quel navigateur (Chrome, Firefox, Edge, etc.).

------------------------------------------------------
3. FONCTIONNALITÉS CLÉS
------------------------------------------------------

* **Logo Vidéo Animé :** Un logo en lecture automatique, silencieux, en boucle, utilisant le format léger WebM (voir code dans le <header>).
* **Quiz Interactif :** Sur la page 'jeux.html', un quiz en JavaScript permet de tester les connaissances sur trois niveaux (Facile, Moyen, Difficile).
* **Moteur de Recherche Filtrable :** Sur la page 'recherche.html', les utilisateurs peuvent filtrer les ressources (outils Open Source) par mots-clés et par thèmes NIRD (Inclusif, Responsable, Durable).

------------------------------------------------------
4. LICENCE & ÉCO-RESPONSABILITÉ
------------------------------------------------------

* **Licence :** Le projet est intégralement Open Source (Licence MIT/GPL, comme mentionné dans le <footer>). Le code est libre d'être consulté, modifié et partagé.
* **Éco-Conception :** L'accent est mis sur l'utilisation de technologies légères (HTML/CSS/JS pur) pour garantir une performance optimale, un faible impact carbone et l'accessibilité sur de vieux matériels.
